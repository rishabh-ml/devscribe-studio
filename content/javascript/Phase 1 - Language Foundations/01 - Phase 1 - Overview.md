# Phase 1 — Language Foundations & Core Syntax

> [!quote] Why This Phase Matters
> Everything you build in JavaScript sits on top of these fundamentals. Shaky foundations create confusion in every later phase. **Do not rush this** — developers who skip or skim this material constantly hit walls when learning closures, async code, and the prototype system.

---

## 🎯 Learning Objectives

By the end of Phase 1, you will:
- Fully understand `var`, `let`, `const` and *why* they behave differently
- Know all 7 primitive types and reference types at a deep level
- Master type coercion — predict what any expression evaluates to
- Use every category of operator confidently
- Distinguish expressions from statements (and know *why* it matters)
- Handle floating-point precision and the `Math` object
- Use `console` methods for effective debugging
