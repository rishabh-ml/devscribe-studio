# Phase 2 — Control Flow, Strings & Basic Data Structures

> [!quote] Why This Phase Matters
> Phase 1 gave you the building blocks — types, variables, operators. Now you learn to **make decisions**, **repeat actions**, and **manipulate JavaScript's core data structures**: strings, arrays, and objects. These are the tools you'll use in literally every JavaScript program you ever write.

---

## 🎯 Learning Objectives

By the end of Phase 2, you will:
- Write branching logic with `if`/`else`, `switch`, and ternaries
- Choose the right loop for every situation
- Manipulate strings with 30+ methods
- Master arrays: creation, mutation, iteration, and transformation
- Work confidently with objects: access, destructure, copy, and compare
- Serialize and deserialize data with JSON
